Năm 4: Web Developer

Ở năm học này, học sinh bắt đầu học cách xây dựng website hoàn chỉnh bằng các công nghệ nền tảng của lập trình web như HTML, CSS và JavaScript.

Các em sẽ hiểu cách hoạt động của website, thiết kế giao diện và xây dựng các trang web tương tác giống như những website thực tế trên Internet.

🌐 Dự án tiêu biểu

  • Website cá nhân
  • Landing page giới thiệu sản phẩm
  • Website portfolio
  • Website nhiều trang

💻 Kỹ năng đạt được

  • HTML & CSS
  • Thiết kế giao diện web
  • Lập trình JavaScript cơ bản
  • Xây dựng website hoàn chỉnh
